The project operating system is under `governance/`.

## Mandatory sequence

```text
Global authority
→ current state
→ one next action
→ applicable specification
→ implementation
→ tests and evidence
→ state reconciliation
```

Start with [AGENTS.md](AGENTS.md), then read [Global Master Plan](governance/GLOBAL_MASTER_PLAN.md).

## Product definition

> TimeProofs verifies, tracks, and preserves evidence of the real outcomes of actions executed by AI agents.

Canonical promise:

> When an AI says “done”, verify what actually happened.

## Global ambition

- product and managed service;

- integrations and developer experience;
- website, brand, SEO, GEO, and AI-first discovery;
- security, privacy, legal, and operations;
