# Comprehension Responses

Place anonymous JSON files downloaded from the local TimeProofs comprehension test in this directory.

Rules:

- do not store names, email addresses, IP addresses, company names, or free-text personal information;
- one response per file;
- never create synthetic responses to improve the report;
- commit only responses the participant agreed could be retained for product research.

Run:

```bash
npm run validation:analyze
```
