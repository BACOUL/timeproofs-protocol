# TimeProofs validation preview

The site is a static, dependency-free, experimental product preview.

It contains:

- four refund Outcome Evidence Packets generated from committed machine bundles;
- an anonymous comprehension test;
- an anonymous structured practitioner workflow test;
- no analytics, cookies, account, contact field, external font, or backend dependency.

## Local preview

From the repository root:

```bash
npm run packets
npm run site:validate
npm run site:serve
```

Open `http://localhost:8080`.

## GitHub Pages

The workflow `.github/workflows/pages.yml` deploys the `site/` directory from `main` after regenerating packets and validating the preview.

The repository owner must create the repository, push `main`, and select **GitHub Actions** as the Pages deployment source once. See `distribution/PREVIEW_DEPLOYMENT_RUNBOOK.md`.

## Evidence handling

The browser sends nothing. Participants download anonymous JSON results and return them through the same channel that shared the preview.

Place accepted files in:

```text
evidence/market/comprehension-responses/
evidence/market/practitioner-feedback/
```

Then run:

```bash
npm run validation:analyze
```

The JSON reports under `site/data/` are generated by the same reference verifier and verdict engine used by the CLI.
